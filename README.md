# 📱 VQueue - Offline Queue Management via Bluetooth LE

> [cite_start]**Showcase Repository**: VQueue is a decentralized mobile application developed as a team project for the **"Application Development for Mobile Devices" (TAMa)** course at **Brno University of Technology (FIT VUT)** during an **Erasmus+ mobility**[cite: 3]. [cite_start]It manages physical queues without any internet connection by utilizing **Bluetooth Low Energy (BLE)** for seamless P2P communication[cite: 3].

## 🚀 Key Features

* [cite_start]**Decentralized Offline System**: Manages virtual queues entirely via BLE, requiring no Wi-Fi or cellular data[cite: 3].
* [cite_start]**Modular Device Roles**[cite: 3]:
    * [cite_start]**Ticket Dispenser**: Allows customers to join a queue and receive a virtual ticket[cite: 3].
    * [cite_start]**Display Screen**: Provides real-time visualization of the current queue status[cite: 3].
    * [cite_start]**Manager Terminal**: Tools for staff to call the next ticket, manage service points, or reset the queue[cite: 3].
* [cite_start]**Real-Time P2P Sync**: Instant synchronization of queue positions across all devices in range[cite: 3].
* [cite_start]**Efficient Discovery**: Optimized BLE protocols for reliable device-to-device connection[cite: 3].
